# DevFolio - Personal Portfolio Website

A sleek, modern, and fully responsive personal portfolio website built carefully from scratch showcasing my skills, projects, and contact information. 

## 🚀 Repository & Live Demo
*   **GitHub Repository Link:** [https://github.com/pranav662/portfolio-website](https://github.com/pranav662/portfolio-website)
*   **Live Website Link (Deployment):** [https://pranav662.github.io/portfolio-website/](https://pranav662.github.io/portfolio-website/)

## ✨ Features
*   **Modern & Premium Design:** Features a dark-themed UI, custom gradient text, glowing micro-animations, and a completely responsive layout designed to look great on any device or screen size.
*   **Skill Arsenal Showcase:** A dynamic diamond grid visualizing technical skills and core competencies.
*   **Project Highlights:** Interactive project cards displaying thumbnails, tech stacks used, and direct links to source code repositories.
*   **Working Contact/Feedback Form:** A form equipped with basic validation that captures user input and effectively saves messages straight to the browser's `localStorage` to reflect the feedback locally.
*   **Clickable Contact Info:** Fully clickable, accessible links pointing to Email addresses and GitHub Profiles, as well as a location redirection to Google Maps.

## 💻 Tech Stack Used
*   **HTML5:** Semantic architecture representing sections and layout.
*   **CSS3:** Extensive use of CSS Variables (Custom properties), Flexbox, CSS Grid matrices, advanced hovering transitions, and keyframe animations for polished visual aesthetics without relying on external libraries.
*   **Vanilla JavaScript (ES6+):** Form validation, seamless DOM element manipulation, preventing XSS injections, and utilizing the Web Storage API (`localStorage`).

## 🛠️ Local Setup & Running
Since this project is purely static and requires no build processes or dependencies, getting started is extremely simple!
1. Clone the repository to your local machine:
   ```bash
   git clone https://github.com/pranav662/portfolio-website.git
   ```
2. Navigate into the directory:
   ```bash
   cd portfolio-website
   ```
3. Open the `index.html` file right in your preferred web browser. Alternatively, serve it via a simple local web server like Live Server (if using VS-Code).
