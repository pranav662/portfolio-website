# Portfolio-Website

## Description
This is a space-themed personal portfolio website built using Next.js and Tailwind CSS. The website includes animated sections such as Hero, About, Skills, Projects, Testimonials, and a Contact form with localStorage-based Transmission Archive system. It demonstrates modern frontend development concepts including routing, state management, responsive design, and interactive UI components.

## Features
- **Space-Themed Interface**: Custom immersive styling.
- **Animated Sections**: Smooth scroll transitions for Hero, About, Skills, Projects, and Testimonials.
- **Transmission Archive**: A fully functional Contact form powered by Javascript `localStorage`.
- **Modern Architecture**: Incorporates advanced state management, component routing, and responsive web design practices.

## Technologies Used
- HTML / CSS / Vanilla JavaScript (Migrating conceptually to Next.js / Tailwind CSS capabilities)
- Intersection Observer API for scroll animations
- LocalStorage API for data persistence
